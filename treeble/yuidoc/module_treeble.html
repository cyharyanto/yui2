Module: Treeble

TreebleDataSource converts a tree of DataSources into a flat list of visible items. This list of items can then be paginated by DataTable. The merged list must be paginated if the number of child nodes might be very large. To turn on this feature, set paginateChildren:true. The total number of items available from each DataSources must remain constant.

This module contains the following classes:

  • TreebleDataSource
  • DataTable
