Class YAHOO.widget.RowExpansionDataTable - extends YAHOO.widget.DataTable

The RowExpansionDataTable class extends the DataTable class to provide functionality for expanding rows to show more contextual data.

oDataSource <YAHOO.util.DataSource> DataSource instance.

collapseAllRows

void collapseAllRows ( )
Collapses all expanded rows. This should be called before any action where the row expansion markup would interfear with normal DataTable markup handling. This method does not remove exents attached during implementation. All event handlers should be removed separately.
